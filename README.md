# Jomivar — Concepto e-commerce (productos en cuero + asistencia IA)

Repositorio de **documentación de idea**: elaboración asistida de una **página web orientada a la venta de productos en cuero**, explorando flujo de trabajo con **GitHub Copilot** u otras herramientas de IA para escribir código.

**Autora del repositorio:** [@Juli21v](https://github.com/Juli21v)

## Estado del repositorio

El contenido principal hoy es la **descripción del proyecto** y la licencia. Es un buen espacio para:

- Añadir mockups, capturas o un prototipo HTML cuando exista
- Enlazar al repo donde viva el código de la tienda

## Sugerencia para el portafolio

Completar el repo con una **carpeta `docs/`** (historias de usuario, wireframes) o el **código fuente** de la tienda para que los reclutadores vean resultado tangible además del concepto.
